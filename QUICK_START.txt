==========================================
STUCK? TROUBLESHOOT:
==========================================
Error: "Port 3000 in use"
→ npm install -g kill-port && kill-port 3000
→ npm start

Error: "API key error"
→ Check .env file exists (not .env.example)
→ API key is pasted correctly
→ Restart server

Error: "Cannot find module"
→ rm -rf node_modules
→ npm install
→ npm start

Error: "CORS" or "Cannot connect"
→ Make sure server is running (npm start)
→ Open http://localhost:3000 (not file://)
→ Check firewall allows port 3000
